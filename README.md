# React Quiz App

A responsive, interactive quiz application built with React. This app presents users with randomized questions, tracks their scores, and features a countdown timer for each question. Instantly see your score and progress as you play!

## Features

- **Randomized Questions**: Each game selects a set of random questions for variety.
- **Multiple Choice Interface**: Clean, intuitive UI for answering questions.
- **Per-Question Timer**: Each question has a 10-second countdown; if time runs out, the quiz skips to the next and awards zero points.
- **Score & Progress Tracking**: Instant feedback after every question and comprehensive results at the end.
- **Live Progress Bar**: Visual tracker of your position in the quiz.
- **Responsive Design**: Works smoothly across desktop and mobile devices.
- **Restart Anytime**: Option to restart the quiz and try again.
- **Easy Customization**: Add or edit questions via a simple JSON file.

## File Structure

- `startscreen.jsx` - Start Screen
- `App.jsx` – Main app logic and state management
- `Quiz.jsx` – Quiz UI with timer functionality
- `QuestionCard.jsx` – Displays each question and options
- `Result.jsx` – Shows final results and score
